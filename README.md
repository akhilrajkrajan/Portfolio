# ESG Portfolio

Personal portfolio website for Akhilraj Krajan — ESG & Sustainability Professional.

Built with Next.js, Tailwind CSS v4, Framer Motion, and TypeScript.

## Deploy

Connect this repository to Vercel. Zero config needed — Next.js is auto-detected.

## Personalize

Edit `lib/data.ts` to update all personal information, experience, projects, and certifications.
